We are building a Fake News Detection System.
Currently, we have implemented a machine learning baseline model using TF-IDF and Logistic Regression, and achieved around 95% accuracy on the IFND dataset.
Next, we plan to extend it using RAG (Retrieval-Augmented Generation) and LLMs for better contextual understanding.
We used the Indian Fake News Dataset (IFND), which contains news statements collected from Indian sources.
The IFND dataset is designed for detecting fake news in the Indian context.
It includes multiple domains such as politics, COVID-19, violence, elections, etc.
This makes it more realistic compared to synthetic or generic datasets.
We selected IFND because:

It is domain-specific (Indian news)
It has real-world data
It is more suitable for building practical applications
We have completed data preprocessing, feature engineering using TF-IDF, and trained a Logistic Regression model achieving around 95% accuracy on the IFND dataset.
This serves as our baseline system.
Next, we aim to enhance the system using RAG and LLMs to improve contextual understanding and real-world performance.
